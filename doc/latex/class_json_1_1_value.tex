\subsection{Detailed Description}
Represents a {\tt J\+S\+O\+N} value. 

This class is a discriminated union wrapper that can represents a\+:
\begin{DoxyItemize}
\item signed integer [range\+: \doxyref{Value\+::min\+Int}{p.}{class_json_1_1_value_a7df8a39e2502b8c92a6a41e3d752d2c8} -\/ \doxyref{Value\+::max\+Int}{p.}{class_json_1_1_value_a978c799a8af3114ef7dab6fd0a310a1b}]
\item unsigned integer (range\+: 0 -\/ \doxyref{Value\+::max\+U\+Int}{p.}{class_json_1_1_value_ac79e63ee68d3aa914bfd6988be669b87})
\item double
\item U\+T\+F-\/8 string
\item boolean
\item \textquotesingle{}null\textquotesingle{}
\item an ordered list of \doxyref{Value}{p.}{class_json_1_1_value}
\item collection of name/value pairs (javascript object)
\end{DoxyItemize}

The type of the held value is represented by a \doxyref{Value\+Type}{p.}{namespace_json_a7d654b75c16a57007925868e38212b4e} and can be obtained using type().

Values of an \doxyref{object\+Value}{p.}{namespace_json_a7d654b75c16a57007925868e38212b4ea6ca35c0a30ea3d1b8ec95c2d1e41a1a8} or \doxyref{array\+Value}{p.}{namespace_json_a7d654b75c16a57007925868e38212b4eaa3025bfd271ef0b0c7c030c9118f8be7} can be accessed using \doxyref{operator[$\,$]()}{p.}{class_json_1_1_value_a7d99f5dba388cdaa152ce6ef933d64ef} methods. Non-\/const methods will automatically create the a \doxyref{null\+Value}{p.}{namespace_json_a7d654b75c16a57007925868e38212b4ea99922f3ccd58446e80e6055a7119b640} element if it does not exist. The sequence of an \doxyref{array\+Value}{p.}{namespace_json_a7d654b75c16a57007925868e38212b4eaa3025bfd271ef0b0c7c030c9118f8be7} will be automatically resized and initialized with \doxyref{null\+Value}{p.}{namespace_json_a7d654b75c16a57007925868e38212b4ea99922f3ccd58446e80e6055a7119b640}. \doxyref{resize()}{p.}{class_json_1_1_value_aa284353271ada427dbfa04a42f2be407} can be used to enlarge or truncate an \doxyref{array\+Value}{p.}{namespace_json_a7d654b75c16a57007925868e38212b4eaa3025bfd271ef0b0c7c030c9118f8be7}.

The \doxyref{get()}{p.}{class_json_1_1_value_a28282c9b76fa031eba7a1843c47c16fe} methods can be used to obtain default value in the case the required element does not exist.

It is possible to iterate over the list of a \doxyref{object\+Value}{p.}{namespace_json_a7d654b75c16a57007925868e38212b4ea6ca35c0a30ea3d1b8ec95c2d1e41a1a8} values using the \doxyref{get\+Member\+Names()}{p.}{class_json_1_1_value_a30fa08af88f2d0a038b22ba9f4e88b2a} method.

\begin{DoxyNote}{Note}
\doxyref{Value}{p.}{class_json_1_1_value_ada6ba1369448fb0240bccc36efaa46f7} string-\/length fit in size\+\_\+t, but keys must be $<$ 2$^\wedge$30. (The reason is an implementation detail.) A \#\+Char\+Reader will raise an exception if a bound is exceeded to avoid security holes in your app, but the \doxyref{Value}{p.}{class_json_1_1_value} A\+P\+I does {\itshape not} check bounds. That is the responsibility of the caller. 
\end{DoxyNote}
